# Aura — Dark AI SaaS Landing Page Template (Tailwind CSS)

A premium, single-file landing page template for AI SaaS startups. Dark mode first, glassmorphism accents, and smooth micro-interactions — no build step required.

## Features

- **Single file** — everything lives in `index.html`
- **Dark mode design** — midnight blues, violet/cyan neon glows, glassmorphism
- **Fully responsive** — mobile, tablet, and desktop
- **No build tools** — Tailwind CSS & Alpine.js via CDN
- **Interactive playground** — typing AI response demo
- **Pricing toggle** — monthly / annual with Alpine.js
- **Mobile menu** — functional hamburger nav

## Sections included

1. Sticky glassmorphism navigation
2. Hero with gradient headline & dashboard mockup
3. Social proof logo bar
4. Bento-style features grid
5. Interactive AI playground
6. 3-tier pricing table (Free / Pro / Enterprise)
7. CTA banner & footer

## Quick start

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/aura-dark-ai-saas-landing-template.git
   cd aura-dark-ai-saas-landing-template
   ```

2. Open `index.html` in your browser — that's it.

   ```bash
   open index.html        # macOS
   xdg-open index.html    # Linux
   ```

No `npm install`. No bundler. No config.

## Customization

| What to change | Where |
|----------------|-------|
| Product name | Search for `AuraAI` in `index.html` |
| Brand colors | `tailwind.config` in the `<script>` block |
| Glow / glass effects | `<style>` block (`.hero-glow`, `.glass`, etc.) |
| Copy & pricing | Edit section HTML directly |
| Fonts | Google Fonts link in `<head>` |

### Color palette (default)

| Token | Value |
|-------|-------|
| Background | `#030712` |
| Surface | `#0a0f1e` |
| Primary accent | Violet `#8b5cf6` |
| Secondary accent | Cyan `#22d3ee` |

## Deploy

### GitHub Pages

1. Push to GitHub
2. **Settings → Pages → Source:** Deploy from branch `main`, folder `/ (root)`
3. Your site will be at `https://YOUR_USERNAME.github.io/aura-dark-ai-saas-landing-template/`

### Netlify / Vercel

Drag and drop the project folder, or connect your GitHub repo. No build command needed.
